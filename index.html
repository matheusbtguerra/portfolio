HTML é a linguagem de marcação padrão usada para criar páginas web. Ela define a estrutura e o conteúdo de uma página, incluindo texto, imagens e links. HTML é essencial para a construção de websites e é facilmente compreensível e implementável.

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a apresentação e o design de páginas web. Ela controla o layout, as cores, as fontes e outros aspectos visuais de um site. CSS permite separar o conteúdo HTML da sua aparência, facilitando a criação de páginas web visualmente atraentes e responsivas.

Java é uma linguagem de programação de propósito geral, popular por sua portabilidade e segurança. É amplamente usada em desenvolvimento de software, desde aplicações empresariais a aplicativos móveis e sistemas embarcados. Java é conhecido por sua robustez, eficiência e grande comunidade de desenvolvedores.

JavaScript é uma linguagem de programação usada principalmente para criar interatividade em páginas da web. Ela permite manipular conteúdo HTML, estilizar elementos com CSS e interagir dinamicamente com os usuários.
